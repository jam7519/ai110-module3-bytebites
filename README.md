ByteBites

ByteBites is a Python-based restaurant ordering system developed as part of the CodePath AI110 course.

Project Overview

This project demonstrates object-oriented programming, software design, testing, and version control using Git and GitHub.

The system allows users to:

Create menu items
Add items to a menu
Create customer orders
Calculate order totals
Filter menu items by category
Validate functionality with automated tests
Project Structure
Files
models.py – Core application classes and business logic
test_bytebites.py – Pytest test suite
bytebites_spec.md – Project specification
bytebites_design.md – UML design documentation
ByteBites_Design_Reference.md – Design notes and references
Features
Menu Management
Add menu items
Store item name, price, category, and rating
Filter items by category
Order Processing
Add menu items to orders
Calculate order totals
Handle empty orders correctly
Testing

The project includes automated tests for:
